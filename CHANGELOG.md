## [v0.1.0] - 2026-05-03

### Added
- 首个公开发布版本
- 支持导入 `.xlsx` 与 `.xls`
- 支持查看原始数据与处理结果
- 支持去除全空行与去重
- 支持按关键词筛选
- 支持按列排序
- 支持按两列做加减乘除生成新列
- 支持柱状图与折线图预览
- 支持导出处理结果为新的 Excel 文件
- 支持新手引导与完整使用说明
- 支持运行日志与异常日志记录
- 提供 Windows 目录版 exe 发布包

### Fixed
- 修复目录版 exe 下 Excel 导入稳定性问题
- 修复 `.xls` 读取链路
- 补强图表中文字体与脏数据处理
- 补强处理、导出、图表相关异常提示

### Notes
- 当前 Release 必须下载并解压完整 zip 包，不能只保留单独 exe
- 当前许可证允许学习与非商用使用，不允许商业使用
