# 桌面数据分析器使用说明

## 应用简介

桌面数据分析器是一款面向本地 Excel 数据的轻量桌面工具，适合在不依赖云端平台、不写代码或只做少量数据清洗分析的场景下快速使用。

## 面向场景

- 运营、销售、财务、人事等岗位查看和整理 Excel 表
- 小团队做本地数据清洗、筛选、排序和结果导出
- 需要快速把 Excel 数据做成基础图表进行核对或汇报
- 对隐私敏感，希望文件只在本机处理，不上传第三方平台

## 面向对象

- 经常使用 Excel，但不想写脚本的业务人员
- 需要快速看数据结果的项目负责人或分析同学
- 想在本地快速验证表格内容的测试、运营或助理岗位

## 功能概览

- 导入 `.xlsx` 和 `.xls` 文件
- 默认读取首个工作表
- 查看原始数据和处理结果
- 去除全空行
- 去除重复行
- 按关键词筛选
- 按列升序或降序排序
- 按两列做加减乘除并生成新列
- 生成柱状图或折线图
- 导出处理结果为新的 Excel 文件
- 记录运行日志，方便定位问题

## 快速开始

1. 启动程序。
2. 点击主工具栏中的“导入 Excel”。
3. 选择本地 `.xlsx` 或 `.xls` 文件。
4. 在左侧查看数据源摘要，确认工作表、行数和列数是否正确。
5. 根据需要勾选去空行、去重、筛选、排序或计算列。
6. 点击“执行处理”。
7. 在“处理结果”标签页查看结果，在右侧查看图表。
8. 点击“导出结果”保存新的 Excel 文件。

## 具体操作步骤

### 1. 导入 Excel

- 点击“导入 Excel”。
- 支持 `.xlsx` 和 `.xls`。
- 程序会自动读取首个工作表。
- 导入成功后，左侧会显示文件名、工作表名、行数和列数。

### 2. 去空行

- 勾选“去除全空行”。
- 点击“执行处理”。
- 所有整行都为空的数据会被移除。

### 3. 去重

- 勾选“去除重复行”。
- 点击“执行处理”。
- 完全相同的重复记录会被保留一条。

### 4. 筛选

- 在“筛选列”中选择一个列。
- 在“筛选关键词”中输入要匹配的文字。
- 点击“执行处理”。
- 程序会保留包含该关键词的记录，不区分大小写。

### 5. 排序

- 在“排序列”中选择一个列。
- 在“排序方式”中选择“升序”或“降序”。
- 点击“执行处理”。

### 6. 计算列

- 在“新列名称”中输入新列名。
- 选择左侧列、运算符和右侧列。
- 支持 `+`、`-`、`*`、`/`。
- 点击“执行处理”后，结果中会新增这一列。

### 7. 查看图表

- 在“图表配置”中选择图表类型。
- 选择“分类列”和“数值列”。
- 右侧会显示柱状图或折线图。
- 如果数值列里有无法转换为数字的内容，这些脏数据会被自动跳过。

### 8. 导出结果

- 点击“导出结果”。
- 选择保存位置。
- 程序会导出当前处理结果为新的 `.xlsx` 文件。

### 9. 重置工作区

- 点击“重置工作区”。
- 当前导入数据、处理结果、输入框和图表都会被清空。

### 10. 新手引导

- 点击主工具栏中的“新手引导”。
- 会先弹出模式选择：
  - 分步引导：逐步介绍每个关键按钮和区域的用途
  - 完整说明：一次性展示完整使用文档

## 常见问题

### 导入失败怎么办？

先确认文件不是损坏文件，也没有被异常程序占用。如果仍然失败，请查看日志文件中的错误信息。

### 为什么有些图表数据没显示？

如果图表数值列里包含空值或非数字内容，这些记录会被自动跳过，避免整张图表报错。

### 导出失败怎么办？

请确认目标 Excel 文件没有被别的软件打开，并且当前目录有写入权限。

### 为什么 exe 看起来没有反应？

优先检查应用目录下的日志文件，确认启动、导入、处理或导出过程中是否出现异常。

## 日志说明

- 日志目录：应用根目录下的 `logs/`
- 日志文件：`logs/runtime.log`
- 日志会记录启动、导入、处理、图表、导出、重置和异常信息
- 状态栏会附带当前日志文件大小，单位为 MB
- 如果要反馈问题，建议同时提供你的操作步骤和相关日志片段
